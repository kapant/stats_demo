Developers_d3js.html : при наведении на любой прямоугольник справа показывается более подробная информация. При нажатии на прямоугольник в начальном состоянии, он добавляется на центральную диаграмму. Кнопки "detailed" и "resize" меняют вид основного графика. "detailed" работает только после перехода из начального состояния.

performance_d3js.html : при наведении на любую точку подсвечиваются все точки в одной дате.

В прототипах есть баги и некоторые недоработки.
